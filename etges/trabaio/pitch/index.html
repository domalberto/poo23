Controle de Gastos

Bruno Etges

Projeto de Aplicação para Disciplina de Programação Orientada a Objetos (06/2023)
Professor Luiz Henrique Rauber Rodrigues
Faculdade Dom Alberto

Problema

  • Muitas vezes gastamos nosso dinheiro como se não houvesse amanhã, só para ficarmos felizes hoje. Mas há um amanhã e há o final do mês também, que é quando nos perguntamos: Para onde foi o meu dinheiro?

Objetivo

Criar uma plataforma que nos permita registrar todos os gastos do mês, para podermos consultar para onde foi o dinheiro.

Metodologia

O aplicativo foi desenvolvido com as tecnologias HTMl, CSS e JavaScript.

  • HTML
  • CSS
  • JS

Resultado

A plataforma ficou com um estilo de planilha, onde é possivel adicionar, remover e verificar os gastos realizados

  • Essas são as categorias de gastos da plataforma
  • Assim é como ficam os registros na plataforma

Futuro

A proxima implementação seria um sistema de login para acesso aos seus gastos.

Uma atualização para levar a plataforma para o mobile, na forma de app.
